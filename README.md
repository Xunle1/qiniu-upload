# qiniu-upload

批量上传图片到七牛云

## 构建项目

环境要求参考：[Wails Installation](https://wails.io/docs/gettingstarted/installation)

### 运行

``` shell
git clone git@github.com:Xunle1/qiniu-upload.git

cd qiniu-upload
```

在 `config.yaml` 设置七牛云相关配置

```shell
make dev
```

### 构建

```
make build
```

应用在 `build/bin` 目录下
